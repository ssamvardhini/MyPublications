# Publications page with citation charts, published live

Goal: build an interactive HTML page of my published papers, with charts of how my citations have
grown over time, and publish it live via GitHub Pages so I can share it.

## Steps
- [x] Confirm ORCID
- [x] Fetch papers from OpenAlex, check the list is really mine
- [x] Design the page (interview, then plan)
- [x] Build index.html
- [ ] Refine the look
- [ ] Publish live with GitHub Pages
- [ ] (Stretch) Package into a reusable `/publications-page` skill

## Design plan (from interview)
- **Style:** academic & classic — serif headings, muted colors, feels like a CV/lab page
- **Accent color:** UW purple (~#4B2E83)
- **Header:** name/title + a one-line bio placeholder: "Software engineer, Skyline & mass spec
  proteomics — University of Washington" (swap in real text later)
- **Featured stat:** total papers (just the one number, kept minimal)
- **Charts:**
  1. Cumulative citations over time (line chart)
  2. Citations per year, not cumulative (bar chart)
  3. Papers published per year (bar chart)
- **Paper list:** simple reverse-chronological list — title (linked to DOI), year, venue, co-authors
  shown under each entry
- **Dark mode:** toggle included
